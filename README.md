# DiscriminantAnalysis.exe 簡易手順書

このツールは、学習ファイルをもとにテストファイルのグループを予測する簡易的な判別分析ツールです。

## 1. 事前準備

次のファイルを用意します。

- 学習ファイル: 正解グループが入っているデータ
- テストファイル: 予測したいデータ

対応形式は `.csv` または `.xlsx` です。

### 学習ファイルの形式

| 列 | 内容 |
| --- | --- |
| 1列目 | ID |
| 2列目以降 | 数値データ |
| 最後の列 | 正解グループ |

正解グループは 2種類以上必要です。

### テストファイルの形式

| 列 | 内容 |
| --- | --- |
| 1列目 | ID |
| 2列目以降 | 数値データ |

学習ファイルとテストファイルで、数値データの列数をそろえてください。

## 2. 実行手順

1. `DiscriminantAnalysis.exe` をダブルクリックします。
2. 「学習ファイル」の参照ボタンから、学習ファイルを選択します。
3. 「テストファイル」の参照ボタンから、テストファイルを選択します。
4. 必要に応じて「出力フォルダ」を選択します。
5. 「実行」ボタンを押します。
6. 完了メッセージが表示されたら終了です。

出力フォルダを指定しない場合は、テストファイルと同じフォルダに結果が保存されます。

## 3. 出力されるファイル

実行すると、主に次の2つのファイルが作成されます。

| ファイル名の例 | 内容 |
| --- | --- |
| `lda_report_日時.csv` または `.xlsx` | 判別結果のレポート |
| `テストファイル名_with_prediction_日時.csv` または `.xlsx` | テストファイルに予測結果を追加したもの |

予測結果ファイルには、最後の列に `predicted_label` が追加されます。

## 4. 注意点

- 数値データの列には、文字ではなく数値を入力してください。
- 学習ファイルの最後の列には、グループ名またはグループ番号を入力してください。
- 学習ファイルとテストファイルの数値データ列数が違うと実行できません。
- 同じ名前の出力ファイルがある場合でも、日時や連番が付くため上書きされません。
